# Predicting Price Moves with News Sentiment (Task 2)

**Task 2** enhances stock price prediction using news sentiment.

## Features

- Data cleaning and merging news sentiment with stock data
- Modular pipeline for processing and feature engineering
- ML models to predict price movement
- Evaluation: Accuracy, Precision, Recall, F1-score

## Structure

```
notebooks/      # Analysis notebooks
scripts/        # Pipeline scripts
tests/          # Unit and integration tests
data/           # Dataset (not included)
requirements.txt # Python dependencies
```

## Install & Run

```bash
git clone -b task-2 https://github.com/Maxd646/news-sentiment-stock-analysis.git
cd news-sentiment-stock-analysis
d pip install -r requirements.txt
python scripts/run_pipeline.py
```
