# Predicting Price Moves with News Sentiment (Task 1)

**Task 1** focuses on initial data exploration and building the base pipeline for predicting stock price movements using historical stock data.

## Features
- Load and clean historical stock price data
- Exploratory data analysis (EDA) and visualization
- Base ML models for price movement prediction
- Evaluation metrics: Accuracy, Precision, Recall, F1-score

## Structure
```
notebooks/      # Analysis notebooks
scripts/        # Data processing and model scripts
tests/          # Unit and integration tests
data/           # Dataset (not included)
requirements.txt # Python dependencies
```

## Install & Run
```bash
git clone -b task-1 https://github.com/Maxd646/news-sentiment-stock-analysis.git
cd news-sentiment-stock-analysis
pip install -r requirements.txt
python scripts/run_pipeline.py
```
